# [InstaCube](https://instacube.moro.es) (instagram-cube)

> Your Instagram feed in a cube!

![InstaCube](cube/images/instagram-cube.jpg "InstaCube")

---

### Install

```sh
$ composer install
$ npm install
```

### Build

```bash
$ gulp create
```

This will create the following files :
	
	index.html
	/cube/css/cube.min.css
	/cube/js/cube.min.js
	/cube/js/vue.min.js
